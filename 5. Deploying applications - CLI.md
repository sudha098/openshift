## ðŸ§­ Deployment Steps (CLI) â€” Using microservices-demo-openshift

```bash

# 2. Apply the full manifest (deploy all services)

```

```
deployment.extensions/carts created
service/carts created
...
```

([GitHub][1])


## ðŸ”„ (Optional) Build DB Images Locally

If you want custom database images (or need to build for a private registry), the repo README suggests building certain DB containers manually using S2I:

```bash
# Example (user-db)
cd docker/user-db-mongodb-container/
s2i build . centos/mongodb-34-centos7 <your-registry>/sockshop-user-db:latest

# Example (catalogue-db)
cd docker/catalogue-db-mysql-container/
s2i build . centos/mysql-57-centos7 <your-registry>/sockshop-catalogue-db:latest
```

Then update the `complete-demo.yaml` (or relevant deployment YAML) to refer to your rebuilt images. ([GitHub][1])
